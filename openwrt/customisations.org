** we disabled uhttp

So there is no symlink in /etc/rc.d anymore:

#+BEGIN_example
openwrt@OpenWrt:~$ sudo /etc/init.d/uhttpd disable

inactive

#+BEGIN_example

lrwxrwxrwx    1 root     root          16 Oct 11 16:01 S50uhttpd -> ../init.d/uhttpd
